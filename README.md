# h5 在线编辑器

## 项目描述
  这是一个在线h5网页制作平台，简洁易上手，没有任何技术门槛，不局限于产品、技术、运营、以及任何想要自己制作h5的小白。

## 代码结构说明
  代码分为制作平台和h5页面二个模块，双入口，入口在src/pages下的admin与view

### 如何运行
```
npm install
```
```
npm run start
```
### 打包
```
npm run build
```
